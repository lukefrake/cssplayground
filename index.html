  • :local-link

    This pseudo-class is called local link pseudo-class and just selects website-internal links. If an positive parameter is passed to this selector, it's possible to select seperated depths of links. These paths are seperated by slashes (/) and scheme, username, password, port, query string and fragment portions of the URL are not considered when selecting.

              :local-link {
              /* declarations */
          }

          :local-link(n) {
              /* declarations */
          }

          /* Would match / link(s) */
          :local-link(0) {
              /* declarations */
          }

          /* Would match /example/ link(s) */
          :local-link(1) {
              /* declarations */
          }

  • :has()

    The :has() selector selects all elements that have one or more elements inside of them, that matches the specified selector.

              :has( selector ) {
              /* declarations */
          }

          li:has( pre ) {
              /* declarations */
          }

  • :not()

    This pseudo-class is called negation pseudo-class which takes as an argument a simple selector list (combination selectors aren't allowed). It matches an element which is not represented by the passed argument and this negations may not be nested within itself like this invalid example: E:not(:not(F))
